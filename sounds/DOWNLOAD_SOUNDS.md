# Sound Files — Download Instructions

All sounds are FREE from **mixkit.co** (no account needed, free license).

Save each file into this `sounds/` folder with the exact filename shown.

---

## How to download from mixkit.co
1. Go to the link
2. Click the **Download Free Sound Effect** button
3. Rename the file to the name in the table below
4. Put it in this `sounds/` folder
